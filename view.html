Información principal

01

¿Qué es HTML?

HTML no es un lenguaje de programación; es un lenguaje de marcado que define la estructura de tu contenido. HTML consiste en una serie de elementos que usarás para encerrar diferentes partes del contenido para que se vean o comporten de una determinada manera.

02

¿Cuál es la importancia de los títulos en HTML?

Los titulos ayudan a darle estructura a un documento. Nos permite organizarlo de manera jerárquica. No solo ayudan a darle organización y estructura, sino qué también permite la accesibilidad y el diseño de la página web.

03

¿Cómo podemos escribir un párrafo en HTML?

Un párrafo es una sección de texto que se agrupa y se separa visualmente de otros bloques de texto. La etiqueta < p > se utiliza para envolver y definir estos bloques de texto.

04

Además de títulos y párrafos, ¿qué otras cosas básicas podemos hacer en HTML?

Los comandos que estas viendo en esta página, son algunos de los muchos que puedes utilizar para la creación de tu página web. Debes tomar en cuenta el hecho de ser consistente y autónomo, esto ayudará a que logres convertirte en el desarrollador que quieres ser.

05

¿Qué son las etiquetas semánticas?

Son elementos que no solo definen la apariencia de una página web, sino también su estructura y significado. Estas etiquetas ayudan a los navegadores y a los motores de búsqueda a entender mejor el contenido de la página, lo que puede mejorar la accesibilidad.

06

Algunos ejemplos de etiquetas semánticas:

Dentro de este apartado, podemos encontrar las siguientes etiquetas: < header > Este elemento nos ayuda a colocar información introductoria. Podemos decir que es donde debemos colocar nuestro menú.

También podemos encontrar < nav > este etiqueta nos ayuda a definir un enlace, pero debemos tener en cuenta que está destinado solo para enlaces de información importantes.

Otra etiqueta importante es: < main > dentro de esta etiqueta, podemos colocar toda la información principal de un documento. Recordemos que no podemos tener más de una en todo nuestro código.

Editores para HTML:

Visual Studio Code

Es un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código.

Atom

Es una aplicación compatible con dispositivos móviles que le ayuda a codificar en su teléfono Android y ejecutarlo sobre la marcha. Editor de texto fácil de usar con funciones de texto sofisticadas. Escriba y guarde código con cualquier extensión

Notepad++

Es un editor de texto y de código fuente libre con soporte para varios lenguajes de programación. Con soporte nativo para Microsoft Windows. Se parece al Bloc de notas en cuanto al hecho de que puede editar texto sin formato y de forma simple.
